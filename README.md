# RAG-бот на базе Ollama
## Описание

Этот проект — Telegram-бот, который использует технологию RAG (Retrieval-Augmented Generation) для ответов на вопросы по файлам.
Бот анализирует документы, извлекает смысловые фрагменты (чанки), ищет релевантные по смыслу части текста и формирует ответ с помощью локальной LLM-модели через Ollama.

## Основные возможности

- Загрузка и обработка файлов (.pdf, .txt, .mp3, .mp4)
- Разделение текста на смысловые чанки
- Создание и хранение эмбеддингов с помощью модели (bge-m3:567m)
- Поиск по смыслу
- Генерация ответов через LLM
- Выбор LLM (gemma3, llama3.1)
- Поддержка контекста (сохранение небольшной истории сообщений)

## Технологии
| Компонент               | Назначение                                           |
| ----------------------- | ---------------------------------------------------- |
| **Python 3.10**         | основной язык                                        |
| **LangChain**           | связывает prompt, LLM и память                       |
| **LangChain-Ollama**    | интеграция с локальной LLM                           |
| **Ollama**              | запуск моделей (`gemma`, `bge-m3:567m`)              |
| **python-telegram-bot** | интерфейс Telegram-бота                              |
| **faster_whisper**      | распознование речи                                   |
| **moviepy**             | работа с видеофайлами                                |
